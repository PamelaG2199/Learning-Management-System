# Learning-Management-System
A full-stack-ready, single-page Learning Management System built with Angular 17 and TypeScript, implementing role-based access control (User, Lead, Admin), a structured multi-tier course approval pipeline, reactive form validation, and dynamic course discovery — architected following SOLID design principles.
